צור בסיס נתונים בשם "מערכת תוכן - הרב" עם הטבלאות והשדות הבאים:

---

טבלה 1: סרטונים
- כותרת — Single line text
- תיאור — Long text
- תאריך — Date
- משך — Single line text
- צפיות — Number (integer)
- קטגוריה — Single select עם האפשרויות: שיעור, הלכה, דרשה, אירוע, ראיון, שונות
- סוג סרטון — Single select עם האפשרויות: youtube, direct
- מזהה יוטיוב — Single line text
- קישור סרטון — URL
- תמונה ממוזערת — URL
- חדש — Checkbox
- מוצג בעמוד הבית — Checkbox

---

טבלה 2: שאלות
- מזהה שאלה — Single line text
- תוכן השאלה — Long text
- שם השואל — Single line text
- קטגוריה — Link to another record (מקושר לטבלת קטגוריות)
- תאריך הגשה — Date
- מפורסם — Checkbox
- הסכמה לפרסום — Checkbox
- סטטוס — Single select עם האפשרויות: ממתין, נענה, מפורסם, נדחה

---

טבלה 3: תשובות

- סוג כותב — Single select עם האפשרויות: רב, מזכירות, אוטומטי
- תאריך — Date

---

תצוגות:
- בכל טבלה צור תצוגת Grid בשם "כל הרשומות"
- בטבלת שאלות צור תצוגה נוספת בשם "ממתין לטיפול" עם פילטר: סטטוס = ממתין
- בטבלת שאלות צור תצוגה נוספת בשם "מפורסם בלבד" עם פילטר: מפורסם = מסומן
- בטבלת סרטונים צור תצוגת Gallery בשם "גלריית סרטונים" עם שדה התמונה: תמונה ממוזערת
- בטבלת יומן אירועים צור תצוגת Calendar בשם "לוח שנה" עם שדה התאריך: תאריך לועזי
